Common CSS Properties

Property Function Values
color Sets element color {text, #hex rbg(), hsl()}
background-color Sets background color {text, #hex, rbg(), hsl()}
background-image Sets image in background {url()}
width Sets element width {px}
min-width Sets minimum width {px}
max-width Sets maximum width {px}
height Sets element height {px}
min-height Sets minimum height {px}
max-height Sets maximum height {px}
display Sets how elements display on page {block, inline-block, inline, none}
margin Sets margins {px}
padding Sets padding {px}
border-width Sets border width {px}
border-style Sets border style {dashed, dotted, double, groove, hidden, inset, outset, ridge, solid, none}
border-radius Sets border curve {px}
position Sets position of element {static, relative, absolute, fixed, sticky}
z-index Sets depth of element {auto, integer}
font-family Sets font type {'Name' if more than one word}
font-weight Sets font thickness {normal, bold, lighter, bolder}
font-style Sets font style {normal, oblique, italic}
opacity Sets element opacity {0.0-1.0}
line-height Sets height of individual lines {px}
word-spacing Sets amount of space between words {px}
